# $YOSUGA - The Narrative of Connection on Base 🧬

$YOSUGA is a community-driven meme project built on **Base**, inspired by the profound concept of "Yosuga" (Bonds/Connections). 

## 🚀 Vision
In the vast sky of Base L2, $YOSUGA aims to redefine the "Sibling Love" and "Forbidden Connection" aesthetics within the Web3 space, merging high-quality ACG (Anime, Comic, Games) culture with decentralized finance.

## 🛠 Technical Stack
- **Network:** Base (Ethereum L2)
- **Standard:** ERC-20 (OpenZeppelin)
- **Frontend:** HTML5 / CSS3 (Glassmorphism UI)

## 📁 Repository Structure
- `/contracts`: Smart contract source code.
- `/website`: Project landing page.
